Evaluating Commands

This specification uses commands that can be used to evaluate an expression.

assertequals

This line is marked green, if the method ForAssertEquals() returns the following string: assertequals.

asserttrue

This line is marked green, if the method ForAssertTrue() returns true.

assertfalse

This line is marked green, if the method ForAssertFalse() returns false.

run

This line is marked green, if the run command executes the AssertEquals specification correctly.
